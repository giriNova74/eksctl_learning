# 🚀 EKS + Kubernetes DevOps Hands-On Repository

This repository demonstrates hands-on implementation of Kubernetes on AWS using eksctl, covering real-world DevOps scenarios from cluster creation to application deployment and operations.

## 🎯 Objective

The goal of this repository is to build practical experience with Amazon EKS (Elastic Kubernetes Service) using eksctl, and to explore core Kubernetes concepts in a cloud-native environment.

This project focuses on learning by doing — simulating real-world DevOps tasks such as cluster provisioning, workload deployment, scaling, networking, and troubleshooting.

---

## 🧱 What This Repository Covers

* Creating and managing EKS clusters using eksctl
* Understanding Kubernetes architecture in AWS
* Working with kubectl for cluster interaction
* Deploying applications to Kubernetes
* Managing Pods, Deployments, and Services
* Scaling applications (manual and auto-scaling)
* Configuring networking and load balancing
* Managing configurations (ConfigMaps & Secrets)
* Monitoring and troubleshooting workloads
* Cleaning up and cost optimization

---

## ⚙️ Tools & Technologies

* AWS (Amazon EKS, EC2, IAM, VPC)
* eksctl (EKS cluster automation)
* kubectl (Kubernetes CLI)
* Docker (containerization)
* Kubernetes (container orchestration)

## 📂 Repository Structure

Each folder represents a specific task or concept:

* `01-cluster-setup/` → Creating EKS cluster using eksctl
* `02-kubectl-basics/` → Interacting with cluster
* `03-deployments/` → Deploying applications
* `04-services/` → Exposing applications
* `05-scaling/` → Scaling workloads
* `06-configurations/` → ConfigMaps & Secrets
* `07-troubleshooting/` → Debugging issues
* `08-cleanup/` → Deleting resources
